Select your operatives according to your team's composition rules. Fixed operatives are included automatically.
Both players share one Crit Op. Select the agreed Crit Op for this game — it determines objective placement and scoring actions.
Crit Op 1
Secure
Control objective markers at the end of each Turning Point. An objective is controlled if you have more operatives within 2″ of it than your opponent.
End of each TP 1VP per controlled objective Max 6VP
Crit Op 2
Loot
Perform the Loot action (1AP) on an objective marker to claim it. Claimed objectives score VP each TP. Opponents can steal your claimed objectives with their own Loot action.
End of each TP 1VP per claimed objective held Max 6VP
Crit Op 3
Transmission
Perform the Transmit action (1AP) while controlling an objective. Each successful transmission scores VP. Each objective can only be transmitted from once per TP.
When Transmit action taken 1VP per transmission Max 6VP
Crit Op 4
Orb
A single Orb objective starts centre-board. Perform Move Orb (1AP) to push it up to 3″ in any direction. From TP2, objectives not holding the Orb score VP for the controlling player.
End of TP2–4 1VP per non-Orb objective controlled Max 6VP
Crit Op 5
Stake Claim
From TP2, each player secretly selects one objective and a condition (Control/Loot/Transmit). If you meet the condition at end of that TP, score bonus VP. Reveal after scoring.
End of TP2–4 2VP if condition met Max 6VP
Crit Op 6
Energy Cells
Pick Up Marker action costs +2AP in TP2, +1AP in TP3, normal cost in TP4. Operatives carrying a marker cannot move more than 6″. Score VP for holding markers and controlling more objectives.
End of each TP 1VP: more objectives controlled 1VP: carrying any marker Max 6VP
Crit Op 7
Download
Perform the Download action (1AP) on the centre or opponent's objective marker — only available in TP3 and TP4. Each objective can only be downloaded from once. Scores VP per completed download.
TP3–4 only 2VP per completed download Max 6VP
Crit Op 8
Data
Secure data caches by controlling objectives with friendly operatives at end of each TP. Score VP for each data cache secured. Caches remain secured even if you lose control later.
End of each TP 1VP per newly secured cache Max 6VP
Crit Op 9
Reboot
At the start of each TP, all objectives go offline (uncontrolled). Players must Reboot (1AP) to bring them back online and then control them. Only rebooted objectives score at end of TP.
End of each TP 1VP per rebooted + controlled objective Max 6VP
